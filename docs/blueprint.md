# **App Name**: Trackly Habits

## Core Features:

- Habit Management: Add, edit, and delete habits with customizable names and descriptions.
- Daily Tracking: Mark habits as completed each day to track progress.
- Streak Counter: Calculate and display the current streak for each habit.
- Data Persistence: Store habit data and progress in Firestore.
- Dashboard Overview: Show current date and habit progress summary in a clean, card-based UI.
- User Authentication: Allow users to sign in using Google Authentication for a seamless experience.
- Suggested Habit: Provide the user with suggested habit that are generated using AI to help them succeed with their habits and give them some new ideas as well as encouragement to succeed.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to create a calming and professional feel.
- Background color: Light gray (#F0F4F8), providing a clean and unobtrusive backdrop.
- Accent color: Pale orange (#FFAD60) for interactive elements and highlights, ensuring affordance and visual interest.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality; suitable for both headlines and body text.
- Use minimalist icons for habit categories and actions. Keep consistency across the app for clarity.
- Employ a card-based layout with proper spacing. Maintain a consistent grid for alignment and readability.
- Incorporate smooth animations on hover and click. Use subtle transitions for user interactions.
